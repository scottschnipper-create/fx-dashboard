# FX Dashboard

Static single-page FX rates dashboard. Vanilla JS + HTML + CSS.

## Stack
- Vanilla JavaScript, HTML, CSS
- No build step — served as static files
- Deployed on Railway via `npx serve`

## Run
```bash
npm start
# or just open index.html locally
```

## Key Files
- `app.js` — All logic (FX data fetching, rendering, charts)
- `index.html` — Page structure
- `style.css` — Styling
- `package.json` — Just the serve script
